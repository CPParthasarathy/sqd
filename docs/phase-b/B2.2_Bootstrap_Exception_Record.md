# B2.2 Bootstrap Merge Exception

## Deviation

Pull request #1 was merged into `main` using a merge commit before the B2.2 squash-only and branch-protection controls were fully active.

Observed commit:

`Merge pull request #1 from CPParthasarathy/docs/b2.2-governance`

## Disposition

The published Git history will not be rewritten or force-pushed.

The deviation is closed by:

1. Enabling squash-only merging.
2. Disabling merge commits and rebase merging.
3. Protecting `main`.
4. Requiring pull requests and linear history.
5. Correcting the verifier.
6. Squash-merging the corrective governance pull request.

This exception authorizes no future merge commits or direct pushes to `main`.
